\# SRE Capstone — Terraform + Kubernetes Connectivity Lab



\## Goal

Build an IaC + Kubernetes lab to practice “why can’t I connect” investigations and produce runbooks/postmortems (Netflix-style support + reliability workflows).



\## Focus

\- AWS networking patterns: SG/CIDR/ports, routing, NAT, hybrid connectivity concepts

\- Kubernetes connectivity debugging: Services, DNS/CoreDNS, Ingress, endpoints



\## Repo Structure

\- infra/terraform/       Infrastructure-as-code (Terraform)

\- k8s/                   Kubernetes manifests (app/services/ingress)

\- docs/runbooks/         Troubleshooting playbooks

\- docs/postmortems/      Incident writeups (templates + examples)



\## Status

In progress — 2 hours/day study sessions (Sun–Thu).
